# Audio Files

Place your audio files here:

## Sound Effects (sfx/)
- pop.mp3 - Button click sound
- success.mp3 - Success/achievement sound  
- listen.mp3 - Microphone activation sound
- correct.mp3 - Correct answer sound
- wrong.mp3 - Wrong answer sound
- star.mp3 - Star collection sound

## Background Music (bgm/)
- menu.mp3 - Menu background music
- lesson.mp3 - Lesson background music
- game.mp3 - Game background music

## Free Sources for Kid-Friendly Audio:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://pixabay.com/sound-effects/

## Recommended Settings:
- Format: MP3
- Bitrate: 128kbps (smaller file size)
- Volume: Normalized
- Length: 0.5-2 seconds for SFX
